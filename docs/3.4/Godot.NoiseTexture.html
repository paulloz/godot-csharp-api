    Class NoiseTexture

    Uses an OpenSimplexNoise to fill the texture data. You can specify the texture size but keep in mind that larger textures will take longer to generate and seamless noise only works with square sized textures.

    NoiseTexture can also generate normalmap textures.

    The class uses Threads to generate the texture data internally, so GetData() may return null if the generation process has not completed yet. In that case, you need to wait for the texture to be generated before accessing the data:

    var texture = preload("res://noise.tres")
yield(texture, "changed")
var image = texture.get_data()
