    Class KinematicCollision

    Contains collision data for KinematicBody collisions. When a KinematicBody is moved using MoveAndCollide(Vector3, Boolean, Boolean, Boolean), it stops if it detects a collision with another body. If a collision is detected, a KinematicCollision object is returned.

    This object contains information about the collision, including the colliding object, the remaining motion, and the collision position. This information can be used to calculate a collision response.
