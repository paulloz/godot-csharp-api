    Class PrimitiveMesh

    Base class for all primitive meshes. Handles applying a Material to a primitive mesh. Examples include CapsuleMesh, CubeMesh, CylinderMesh, PlaneMesh, PrismMesh, QuadMesh, and SphereMesh.

    CustomAabb

    Overrides the AABB with one defined by user for use with frustum culling. Especially useful to avoid unexpected culling when using a shader to offset vertices.

    FlipFaces

    If set, the order of the vertices in each triangle are reversed resulting in the backside of the mesh being drawn.

    This gives the same result as using Back in ParamsCullMode.

    GetMeshArrays()

    Returns mesh arrays used to constitute surface of Mesh. The result can be passed to AddSurfaceFromArrays(Mesh.PrimitiveType, Array, Array, UInt32) to create a new surface. For example:

    var c := CylinderMesh.new()
var arr_mesh := ArrayMesh.new()
arr_mesh.add_surface_from_arrays(Mesh.PRIMITIVE_TRIANGLES, c.get_mesh_arrays())
