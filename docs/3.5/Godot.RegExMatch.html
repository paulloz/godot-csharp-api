    Class RegExMatch

    Contains the results of a single RegEx match returned by Search(String, Int32, Int32) and SearchAll(String, Int32, Int32). It can be used to find the position and range of the match and its capturing groups, and it can extract its substring for you.

    GetEnd(Object)

    Returns the end position of the match within the source string. The end position of capturing groups can be retrieved by providing its group number as an integer or its string name (if it's a named group). The default value of 0 refers to the whole pattern.

    Returns -1 if the group did not match or doesn't exist.
