    Class Dictionary<TKey, TValue>

    Typed wrapper around Godot's Dictionary class, a dictionary of Variant typed elements allocated in the engine in C++. Useful when interfacing with the engine. Otherwise prefer .NET collections such as System.Collections.Generic.Dictionary`2.
