    Class EditorInspectorPlugin

    EditorInspectorPlugin allows adding custom property editors to EditorInspector.

    When an object is edited, the CanHandle(Object) function is called and must return true if the object type is supported.

    If supported, the function ParseBegin(Object) will be called, allowing to place custom controls at the beginning of the class.

    Subsequently, the ParseCategory(Object, String) and ParseProperty(Object, Int32, String, Int32, String, Int32) are called for every category and property. They offer the ability to add custom controls to the inspector too.

    Finally, ParseEnd() will be called.

    On each of these calls, the "add" functions can be called.

    To use EditorInspectorPlugin, register it using the AddInspectorPlugin(EditorInspectorPlugin) method first.
